# Task 13: Even or Odd Number Checker

## Overview
This task demonstrates how to check whether a number is even or odd using the modulo operator (`%`) in JavaScript.

## Objective
Determine if given numbers are even or odd by:
- Using the modulo operator to find the remainder when dividing by 2
- Implementing conditional statements (if/else) to check the result

## How It Works

### Even Number Check
- If a number modulo 2 equals 0, the number is **even**
- Example: `5 % 2 == 0` → `false` (5 is odd)

### Odd Number Check  
- If a number modulo 2 is not equal to 0, the number is **odd**
- Example: `10 % 2 == 0` → `true` (10 is even)

## Files

- **index.html** - Basic HTML structure with script reference
- **script.js** - JavaScript code containing the even/odd checking logic

## Test Cases

The script tests two numbers:
- `firstNumber = 5` → Output: "firstNumber is odd and is 5"
- `secondNumber = 10` → Output: "secondNumber is even and is 10"

## Execution

To run this program:
1. Open `index.html` in a web browser
2. Open the browser's Developer Console (F12)
3. View the console output showing whether each number is even or odd

## Key Concepts
- **Modulo Operator (%)** - Returns the remainder of a division
- **Conditional Statements (if/else)** - Used to make decisions based on conditions
- **Console Logging** - Used to display output in the browser console
